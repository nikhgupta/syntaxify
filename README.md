# Syntaxify

Quickly syntax highlight a Github URL.

    GET /<mode>/?url=<url>&theme=<theme>

### Available Themes

- thankful_eyes
- colorful
- base16
- base16.dark
- base16.solarized
- base16.solarized.dark
- base16.monokai
- base16.monokai.light
- github
- monokai
- molokai
- monokai.sublime
